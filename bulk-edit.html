② อัพโหลดไฟล์ที่แก้แล้ว

📂 ลากไฟล์ Excel (.xlsx / .csv) มาวางที่นี่ หรือคลิกเพื่อเลือกไฟล์
📌 กติกา: คอลัมน์ sku = ตัวจับคู่แถว ห้ามแก้ · ช่องว่าง = ไม่แก้ (คงค่าเดิม) · พิมพ์ x = ล้างค่า (เช่น ล้างราคาโปรฯ) · คอลัมน์ name มีไว้ดูเฉยๆ ระบบไม่เอาไปเขียน · ลบแถวทิ้ง = ข้ามสินค้านั้น
